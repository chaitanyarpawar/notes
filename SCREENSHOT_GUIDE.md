## ğŸ“² How to Take Screenshots

### Method 1: Physical Device
1. Prepare the screen with content
2. Press **Power + Volume Down** simultaneously
3. Screenshot saved to Gallery/Photos
4. Transfer to computer via USB

### Method 2: Android Studio Emulator
1. Open emulator with desired resolution
2. Set up app with sample data
3. Click camera icon in emulator toolbar
4. Screenshots saved to computer

### Method 3: ADB Command
